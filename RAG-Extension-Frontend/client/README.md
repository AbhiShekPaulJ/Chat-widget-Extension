# RAG Assistant – Chrome Extension

**RAG Assistant** is a Chrome extension that leverages Retrieval-Augmented Generation (RAG) with Azure OpenAI to help users interact with webpage content. Users can ask questions, summarize, or extract key points from the entire page or any selected text.

## Features

- **Ask Questions:** Get answers about the current webpage.
- **Summarize:** Generate concise summaries of the page or selected text.
- **Key Points:** Extract bullet-point key points from content.
- **Contextual Selection:** Only the selected text is sent to the AI if the user highlights part of the page.
